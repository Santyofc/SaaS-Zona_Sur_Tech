# Facturación Electrónica en Costa Rica: Guía Completa 2025

La facturación electrónica en Costa Rica es obligatoria para todas las empresas y personas físicas con actividad económica. El **Ministerio de Hacienda** exige el uso de comprobantes electrónicos versión **4.3** desde 2023.

## ¿Qué es la Facturación Electrónica?

Es el proceso de emisión, transmisión y almacenamiento de documentos comerciales en formato XML, firmados digitalmente y validados por el Ministerio de Hacienda de Costa Rica.

## Tipos de Comprobantes Electrónicos

### 1. Factura Electrónica
Para ventas a contribuyentes del impuesto. Incluye el número de cédula del receptor.

### 2. Tiquete Electrónico
Para ventas al consumidor final cuando no se requiere identificación.

### 3. Nota de Crédito Electrónica
Para devoluciones, descuentos y anulaciones de facturas previas.

### 4. Factura de Exportación
Para ventas al exterior exoneradas de IVA.

## Requisitos Técnicos

- **Firma digital** emitida por el Banco Central de Costa Rica (BCCR)
- **Certificado digital** vigente
- **Conexión a internet** para validación en tiempo real con Hacienda
- **Software homologado** o certificado para emisión de XML versión 4.3
